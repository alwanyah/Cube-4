# Cube^4
![Screenshot (323)](https://user-images.githubusercontent.com/95567513/188308814-05080ccb-e99b-4e93-9d00-ece632c6b79b.png)
3D game where you should avoid to colliding with other obstacles

## control
- ´A´ to go left
- ´D´ to go right

## installation
1. go to ´Builds´
2. install ´Cube^4.zip´
3. extract the files
4. start ´Cube^4.exe´
5. have fun :)
